## 操作說明

### 快捷鍵
| 按鍵 | 功能 |
|------|------|
| `P` | 切換投資組合 |
| `A` | 新增交易 |
| `R` | 查看再平衡建議 |
| `H` | 幫助 |
| `Q` | 離開 |
| `Esc` | 返回/取消 |
| `Enter` | 確認 |
| `↑/↓` | 選擇項目 |
| `Tab` | 切換欄位 |

## 色彩方案

### 價格變化
- 🟢 正報酬：綠色
- 🔴 負報酬：紅色
- ⚪ 持平：灰色

### 按鈕狀態
- 選中：藍底白字
- 未選：灰底白字
